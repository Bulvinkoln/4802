Инструкция по отправлению на гит
1. На гите создаём репо
2. Создаём папку  в терминале
3. инит гит
4. add - commit
5. push

Инструкция по работе с удалённым репозиторием

1. Сделать форк
2. Создать папку и в терминале сделать клоне ссылки
3. через cd выбрать папку (если необходимо)
4. Создать файл ( windows): type nul > example.txt
5. Сохранить

Словарь
1. git init
2. git add . в текущем каталоге
3. git add "name"
4. git status изменения ( локально)
5. git commit -m
6. git branch ветки
7. git checkout переход на ветку
8. git push отправить
9. git pull выгрузить и объединить
10. git diff отличия текущей и сохранненой ветки
11. git merge объединить
12. git reset HEAD^1 удалить последний коммит
